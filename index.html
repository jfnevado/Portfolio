Recoleción de datos:

Tengo experiencia en recolectar y extraer datos de varias fuentes, incluyendo bases de datos, APIs y web scraping. He trabajado con herramientas como Python, SQL y BeautifulSoup para acceder a los datos y automatizar el proceso de recolección.

Limpieza y transformación:

La limpieza y el procesamiento de datos son una parte fundamental de mi trabajo. Tengo experiencia utilizando herramientas como Excel, Python y SQL para realizar tareas como la eliminación de valores atípicos, la imputación de datos faltantes y la normalización de datos.

Visualización:
